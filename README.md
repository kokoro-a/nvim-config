# KZ neovim

## Install

### Prerequisites

- Neovim >= v0.8.0

```sh
  brew install neovim
```

- Lazygit
    
```sh
  brew install lazygit
```

- for telescope.nvim

```sh
  brew install ripgrep fd
```

### Install
```sh
  # required
  mv ~/.config/nvim{,.bak}

  # optional but recommended
  mv ~/.local/share/nvim{,.bak}
  mv ~/.local/state/nvim{,.bak}
  mv ~/.cache/nvim{,.bak}

  git clone https://github.com/siawkz/nvim ~/.config/nvim 
```

### Recommended Fonts

JetBrainsMono Nerd Font
